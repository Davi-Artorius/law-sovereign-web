# ⚖️ Law Sovereign — CRM para Escritórios de Advocacia

Sistema de gerenciamento de relacionamento com clientes (CRM) construído especificamente para escritórios de advocacia, automatizando qualificação de clientes, acompanhamento de processos e comunicação multi-canal.

## 🏛️ Stack

- **Frontend:** React 19 + TypeScript + Vite + Tailwind CSS v4
- **Backend:** Node.js + Express (em desenvolvimento)
- **Database:** PostgreSQL + Prisma ORM
- **Integrations:** WhatsApp Business API, Email, SMS
- **Infrastructure:** Docker, CI/CD ready

## 🚀 Funcionalidades

- 📋 **Gestão de Clientes** — Pipeline visual com histórico e qualificação automática
- 📞 **Multi-Canal** — WhatsApp, Email, SMS integrados
- 📊 **Dashboard Analytics** — Métricas em tempo real
- 🤖 **Automações** — Follow-ups inteligentes
- 📁 **Documentos** — Gestão por cliente/processo
- 🔐 **Compliance** — Auditoria completa

## 🚀 Como Rodar

```bash
git clone https://github.com/Davi-Artorius/law-sovereign-web
cd law-sovereign-web
npm install
npm run dev
```

Abre http://localhost:5173
